Drawing Tips

  • Each frame is 32x32px. Pixel art tools recommended (Aseprite, Piskel, Pixilart, etc.).
  • Corgi features: short legs, long body, big ears, fluffy butt
  • Side views (E/W) show off the long body. Front view (S) works well as a sitting pose.
  • Background must be transparent.
  • Combine all frames into a single PNG and save as corgi.png.
  • Position for [col, row] = col * 32px from left, row * 32px from top
